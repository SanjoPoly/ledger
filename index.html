Use Ledgr on all your devices

Easiest way: download the setup file below and upload it to your website repo (the one serving this page) — top level, next to index.html. Every device and browser will then open straight at the sign-in screen. The file holds your connection encrypted with your password; only your password can unlock it, so use a strong one.

Upload steps: open your website repo on GitHub → Add file → Upload files → drop in ledgr-config.json → Commit. Wait a minute, then test in an incognito window.

If you ever change your password, generate and upload a fresh setup file — the old one only opens with the old password. To revoke everything, delete the token in GitHub → Developer settings.
